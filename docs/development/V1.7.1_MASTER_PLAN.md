### Why Open Source?

**Original Plan**: FastReactCMS (public) + TheITApprenticeLMS (private dual-repo)

**New Strategy**: Single open-source repo with plugin system

**Reasons**:
1. **Build in Public** - Demonstrate skills publicly, not hide them
2. **Portfolio Value** - One impressive platform > two separate projects
3. **Community Growth** - Open source LMS attracts contributors
4. **No Maintenance Hell** - No syncing between two repos
5. **Market Positioning** - "Open-source CMS that becomes full LMS" is unique
6. **Flexibility** - Plugins can be disabled = pure blog OR full LMS

**What Stays Private**:
- Production database (user data, content)
- API keys (.env files)
- Custom course content you create
- User progress data

**What Goes Public**:
- All source code (backend + frontend)
- Plugin architecture
- Admin builders (tutorial editor, quiz builder, etc.)
- XP/Achievement system
- Documentation

**Branding**:
- **FastReactCMS** = Core brand (blog, pages, media)
- **FastReactCMS LMS Edition** = Full platform with learning plugins enabled
- **Tagline**: "Modern CMS that transforms into a full LMS with one click"
